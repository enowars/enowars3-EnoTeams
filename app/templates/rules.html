Rules

We'll all survive if we stick to some rules...

Rule #1: DO NOT ATTACK THE INFRASTRUCTURE

We are all humans (even students) and we all make mistakes... Do not exploit any bug you discovered in the infrastructure. Rather report it to the organizers. We reserve the right to award exclusive bonus points depending on the bug and its severity.

Rule #2: NO (D)DOS

Extensive bandwidth usage and/or attacks leading to the DoS of other teams or the infrastructure will be punished with a permanent ban. You will not win using this kind of attacks.

Rule #3: Prices

A surprise trophy for the first three places ;)

Rule #4: The scoring formula

We are currently using the scoring formula by Faust CTF. Thanks alot for making that public.

Rule #5: Escape Clause

We preseve the right to change any rule, mode of competition, or simply anything about this event in any way.

Rule #6: Flag submission

The Flagsubmission is done via a tcp connection to host 49.12.115.199 on port 1337. There you can just send the flag or multiple flags delimited by \n characters. Each Flag will be checked by the backend and then a message is returned picked by this code:

  • FlagSubmissionResult.Ok => "VALID: Flag accepted!",
  • FlagSubmissionResult.Invalid => "INVALID: You have submitted an invalid string!",
  • FlagSubmissionResult.Duplicate => "RESUBMIT: You have already sent this flag!",
  • FlagSubmissionResult.Own => "OWNFLAG: This flag belongs to you!",
  • FlagSubmissionResult.Old => "OLD: You have submitted an old flag!",
  • FlagSubmissionResult.UnknownError => "ERROR: An unexpected error occured :(",
  • FlagSubmissionResult.InvalidSenderError => "ILLEGAL: Your IP address does not belong to any team's subnet!",
  • FlagSubmissionResult.SpamError => "SPAM: You should send 1 flag per line!",
